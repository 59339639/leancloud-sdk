# AVOS Cloud简介

AVOS Cloud(中文名:万象云)提供移动服务端整体解决方案，为移动应用开发者提供稳定可依赖的后端云服务，包括存储、账号管理、社交分享、推送等以及相关的技术支持和服务。

详情请访问我们的[官方网站](https://cn.avoscloud.com/)。
